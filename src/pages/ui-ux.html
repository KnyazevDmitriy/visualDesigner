Quest of Words

Mobile app design

A mobile puzzle game in which the player needs to form words from the given letters in a circle.

The design of a mobile game aimed at developing and maintaining the level of logical thinking of an adult. The mechanics of the game is a riddle, where the user finds among a certain set of letters. For 10 riddles, the same set of letters is presented, in which you need to find all the words. The game also features different levels of difficulty. The design is chosen calm, so that the user can relax and enjoy the time in the game. The game also has a relaxing character for resting the brain from constant trifling thoughts. The interface and game mechanics help improve concentration.
